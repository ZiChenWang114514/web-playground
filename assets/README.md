# assets/

Put your personal homepage static files here:

- `avatar.jpg` — your photo (square, ≥ 400×400 recommended). After dropping it in,
  open `profile.config.js` and change `avatar: "assets/avatar-placeholder.svg"` → `"assets/avatar.jpg"`.
- `cv.pdf` — your CV. Already referenced in `CONTACTS` by default.
- `avatar-placeholder.svg` — auto-generated fallback, do not delete unless you have a real avatar ready.

All files here are served as-is via GitHub Pages.
